# Sistema Gráfico Interativo

Projeto de um SGI para a disciplina de Computação Gráfica (INE5420).

***

### Build & Run

Recomendado criar uma pasta separada para a build:
`mkdir build`
`cd build`

Compilar com:
`cmake ../src`
`make`

Executar:
`./gtk-renderer`
